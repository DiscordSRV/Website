# Website

DiscordSRV's website. Built with [Next.js](https://nextjs.org/)

### Develop

- Install dependencies:
```bash
npm install
# or
yarn install
```

- Run the development server:
```bash
npm run dev
# or
yarn dev
```
- Open [http://localhost:3000](http://localhost:3000) with your browser.
